# Install

**Host board (Pi 4/5, Jetson, x86):**
```bash
curl -fsSL get.tafy.sh | bash
```

Then open https://tafy.local on your laptop.

Add a node:
```bash
curl -fsSL get.tafy.sh | bash -s -- --join <JOIN-CODE>
```

> Prefer a prebuilt image? See /docs/install-images (coming soon).
